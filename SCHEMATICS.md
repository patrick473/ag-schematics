# Schematics Reference

| Schematic | Description |
|-----------|-------------|
| `header` | Generates an Angular header component |
| `adr` | Generates an Architecture Decision Record (ADR) markdown file |
| `docker-config` | Adds a `Dockerfile` and `compose.yaml` to the project |
| `gitignore-config` | Adds a `.gitignore` file to the project |
| `k8s-config` | Adds Kubernetes Deployment, Service, and Ingress manifests |
| `lint-config` | Adds ESLint, Prettier, and EditorConfig configuration files |
| `npm-config` | Adds `.npmrc` and `.nvmrc` configuration files |
| `orval-config` | Adds [Orval](https://orval.dev) with a config file and empty OpenAPI spec |
| `otel-config` | Adds OpenTelemetry Collector configuration and nginx OTLP proxy setup |
| `sonar-config` | Adds SonarQube configuration |
| `ado-config` | Adds an Azure DevOps default pull request template to the project |
